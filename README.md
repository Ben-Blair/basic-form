# Forms Exercise

Proposal form (labels and styling) — Assignment 5.

## Repository structure (for GitHub)

Mirror this structure when you upload:

```
forms/                          (or "basic-form" — your chosen subfolder)
├── form-exercise1(one).html    (main form page)
├── css/
│   └── form.css                (stylesheet)
└── README.md
```

The HTML file links to the stylesheet with:

```html
<link rel="stylesheet" href="css/form.css" />
```

So the stylesheet must live in a subdirectory named `css` as `form.css`. If you rename the file or the folder, update this `href` in the `<head>` of `form-exercise1(one).html`.

## Running locally

Open `form-exercise1(one).html` in a browser. The form will use `css/form.css` as long as the `css` folder sits next to the HTML file.
